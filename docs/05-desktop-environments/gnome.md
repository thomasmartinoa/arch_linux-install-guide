## üì¶ Installation

### Full Installation

```bash
sudo pacman -S gnome gnome-extra
```

### Minimal Installation

```bash
sudo pacman -S gnome-shell gnome-control-center gnome-terminal nautilus gdm
```

### Enable Display Manager

```bash
sudo systemctl enable gdm
```

### Reboot

```bash
sudo reboot
```
